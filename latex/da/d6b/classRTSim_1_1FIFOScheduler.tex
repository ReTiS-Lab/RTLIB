\subsection{Detailed Description}
This class implements a simple First in First out scheduling policy. In other words, tasks are order by their arrival time. It redefines only the add\+Task function, because most of the work is done in the \hyperlink{classRTSim_1_1Scheduler}{Scheduler} class.
