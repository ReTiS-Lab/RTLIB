\subsection{Detailed Description}
Abstract \hyperlink{classRTSim_1_1Task}{Task} interface. This interface is common to all objects that will implement something similar to a task, i.\+e. anything that can be scheduled by a kernel. For example, both \hyperlink{classRTSim_1_1Task}{Task} and \hyperlink{classRTSim_1_1Server}{Server} implement this interface.

Therefore, this interface defines all methods that are essential for a kernel to schedule a task.
