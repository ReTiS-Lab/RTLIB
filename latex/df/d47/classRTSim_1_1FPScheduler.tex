\subsection{Detailed Description}
This class implements a generic fixed priority scheduler.

This class redefines only the add\+Task function, because most of the work is done in the \hyperlink{classRTSim_1_1Scheduler}{Scheduler} class. The user must specify the priority of each task esplicitily.
