\subsection{Detailed Description}
The RT \hyperlink{classRTSim_1_1Task}{Task} Set factory. For a Real-\/\+Time \hyperlink{classRTSim_1_1Task}{Task} Set we need to update/modify the main generation loop, because we need to generate A\+L\+SO the deadline and the virtual task constructor

It has two constructors\+: one for tasks without offset (i.\+e. with offset 0), and the other for task with offset. In the first case, in addition to the computation time generator and the interarrival time generator, the user must specify the relative deadline generator. In the second case, the user must also specify the offset generator.
