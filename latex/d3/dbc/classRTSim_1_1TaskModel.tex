\subsection{Detailed Description}
Contains the scheduling parameters and a pointer to the task. It is used by a scheduler to store the pointer to the task and the set of scheduling parameters.

Each scheduler has its own task model. So the class inheritance trees of the task models and of the schedulers are similar.
