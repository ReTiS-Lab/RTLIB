\subsection{Detailed Description}
This class implements the Priority Inheritance Protocol. It has the same interface of any \hyperlink{classRTSim_1_1Resource}{Resource} Manager class, in addition it has a set\+Scheduler() function, because the PI needs the scheduler for changing the priority of a task.
