Detailed Description

CPU implementation by Claudio Scordino. A CPU doesn't know anything about who's running on it: it just has a speed factor. This model contains the energy values (i.e. Voltage and Frequency) of each step. The speed of each step is calculated basing upon the step frequencies. The function setSpeed(load) adjusts the CPU speed accordingly to the system load, and returns the new CPU speed.
